# AXIOMBYTE SMS Database

Run these files in order:

1. `schema.sql`
2. `seed.sql`

## Main Links

- `programmes` feed Learning Area fields.
- `subjects` link to programmes.
- `classes` link to programmes.
- `class_subjects` supports 12 subject options per class.
- `students` link to classes and houses.
- `student_subjects` stores each student's selected subjects.
- `staff_users` stores school administrators and staff.
- `documents` stores titled uploads for students and staff.
- `assessments` stores the capture-assessment header.
- `assessment_scores` stores each student's mark, grade, and remark.

## Assessment Modes

The schema seeds the five assessment modes currently used on `cass.html`.
